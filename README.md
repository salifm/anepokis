# Anepokis

> STATUS: Pre-alpha

## Install

```fish
go get -u salifm.com/go/anepokis
```

## Run

```fish
anepokis 8080
```

and navigate to [salifm.com/anepokis](https://salifm.com/anepokis) or [localhost:8080/docs](http://localhost:8080/docs)
